# Análisis de conjunto de datos de vehículos para los anuncios de las ventas de coches

En este pequeño proyecto podremos visualizar graficos que explican como se comportan los datos segun
la cantidad de kilometraje registrada en el Odómetro de los coches cuya información es util para la
venta de autos usados. 


Link de la aplicación Render: https://tripleten-module7-project.onrender.com
